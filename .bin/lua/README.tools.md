
Tools Executables Downloads
---------------------------

Here you will find packages of the executables of the tools described in the documentation.

These tools includes the standalone interpreter that can be used to run Lua code.

Available Platforms
-------------------

Here is a list of packages and some details of the platforms where they were built:

* **Linux26g4:** Ubuntu 10.04 (x86) / Kernel 2.6 / gcc 4.4 / GTK 2.20
* **Linux26g4_64:** Ubuntu 10.04 (x64) / Kernel 2.6 / gcc 4.4 / GTK 2.20
* **IRIX6465:** SGI IRIX 6.5 (mips) / gcc 3.3 / Motif 1.2
* **SunOS510:** Sun Solaris 10 (sparc) / gcc 3.4 / Motif 2.1 
* **MacOS106:** Mac OS X 10.6 (x64) [Snow Leopard] / Darwin Kernel 10 / gcc 4.2 / Fink GTK 2.18
* **Win32:** Executables only for Windows NT/2000/XP/Vista/7 (generated by vc8)
* **Win64:** Same as Win32 but for 64-bits systems using the x64 standard
* **Win32_cygw17:** Executables only for Windows NT/2000/XP, but using the Cygwin Posix system
